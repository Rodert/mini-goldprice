## 🎯 设计特点

### 1. 配色方案
- **主色调**：金色 `#FFD700`
- **辅助色**：橙色 `#FFA500`
- **成功色**：绿色 `#67c23a`
- **警告色**：黄色 `#e6a23c`
- **危险色**：红色 `#f56c6c`
- **信息色**：灰色 `#909399`

### 2. UI 风格
- 基于 **Element UI** 设计规范
- 现代扁平化风格
- 卡片式布局
- 金色渐变主题

### 3. 响应式设计
- 支持 PC 端（1920x1080）
- 支持平板端（1024x768）
- 使用 Grid 布局自适应

### 4. 交互设计
- Hover 悬浮效果
- 点击按钮提示（alert/confirm模拟）
- 状态标签颜色区分
- 图标语义化

## 📝 使用说明

### 查看原型
1. 打开 `index.html` 浏览所有页面
2. 点击对应卡片进入具体页面
3. 点击"返回导航"回到首页

### 编辑原型
1. 所有页面使用纯 HTML + CSS
2. 通用样式在 `common.css`
3. 特殊样式写在页面内 `<style>` 标签中
4. 交互使用简单 JavaScript（alert/confirm）

### 导出设计
- 可直接截图作为设计稿
- 可复制 HTML 结构到 Vue 组件
- 可提取 CSS 样式到实际项目

## 📌 注意事项

1. **浏览器兼容**：建议使用 Chrome/Edge/Safari 最新版
2. **Emoji 显示**：部分 Emoji 在不同系统显示效果略有差异
3. **交互模拟**：原型中的按钮点击使用 alert/confirm 模拟，实际开发需替换
4. **数据真实性**：所有数据为示例数据，实际项目需对接后端 API
