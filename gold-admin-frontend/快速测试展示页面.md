# 快速测试金价展示页面

## 方法一：使用在线占位资源（推荐用于测试）

如果暂时没有视频和图片资源，可以修改代码使用在线占位资源。

### 修改步骤

编辑 `src/views/display/index.vue`，找到以下代码：

**修改视频地址：**
```javascript
// 将这一行：
videoUrl: '/assets/jewelry-video.mp4',

// 改为使用在线视频或注释掉：
videoUrl: '', // 暂时不显示视频
```

**修改产品图片地址：**
```javascript
// 将这一段：
productImages: [
  '/assets/product1.jpg',
  '/assets/product2.jpg',
  '/assets/product3.jpg',
  '/assets/product4.jpg',
  '/assets/product5.jpg'
],

// 改为使用占位图片服务：
productImages: [
  'https://via.placeholder.com/400x400/FFD700/000000?text=Product+1',
```

## 方法二：使用本地资源文件

### 1. 准备视频文件

找一个珠宝或商品展示的视频，命名为 `jewelry-video.mp4`，放到：
```
gold-admin-frontend/public/assets/jewelry-video.mp4
```

**视频要求：**
- 格式：MP4 (H.264)
- 建议尺寸：1920x1080 或 1280x720
- 建议时长：20-60秒

**获取免费视频资源：**
- Pexels: https://www.pexels.com/zh-cn/search/videos/jewelry/
- Pixabay: https://pixabay.com/zh/videos/search/jewelry/

### 2. 准备产品图片

准备至少5张产品图片（JPG/PNG格式），命名为：
- product1.jpg
- product2.jpg
- product3.jpg
- product4.jpg
- product5.jpg

放到：
```
gold-admin-frontend/public/assets/
```

**获取免费图片资源：**
- Unsplash: https://unsplash.com/s/photos/jewelry
- Pexels: https://www.pexels.com/zh-cn/search/jewelry/

### 3. 使用命令行快速下载（可选）

如果你的系统支持 curl，可以使用以下命令快速创建测试图片：

```bash
cd gold-admin-frontend/public/assets/

# 下载占位图片（需要网络）
curl "https://via.placeholder.com/400x400/FFD700/000000?text=Product+1" -o product1.jpg
curl "https://via.placeholder.com/400x400/C0C0C0/000000?text=Product+2" -o product2.jpg
curl "https://via.placeholder.com/400x400/CD7F32/000000?text=Product+3" -o product3.jpg
curl "https://via.placeholder.com/400x400/FFD700/000000?text=Product+4" -o product4.jpg
curl "https://via.placeholder.com/400x400/C0C0C0/000000?text=Product+5" -o product5.jpg
```

## 启动和测试

### 1. 确保后端服务运行

```bash
cd gold-admin-backend
go run main.go
```

### 2. 启动前端服务

```bash
cd gold-admin-frontend
npm run serve
```

### 3. 访问展示页面

打开浏览器访问：
```
http://localhost:8080/display
```

按 `F11` 进入全屏模式，获得最佳展示效果。

## 如果遇到问题

### 视频不显示
- 检查文件路径是否正确
- 检查视频格式是否为 MP4
- 暂时设置 `videoUrl: ''` 跳过视频

### 图片不滚动
- 检查图片文件是否存在
- 使用在线占位图片测试
- 检查浏览器控制台错误

### 金价数据为空
- 检查后端服务是否运行
- 检查是否有价格数据在数据库
- 页面会自动使用模拟数据作为备份

### 动画不流畅
- 尝试使用 Chrome 浏览器
- 检查电脑性能
- 关闭其他占用资源的程序
