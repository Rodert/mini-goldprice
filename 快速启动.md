## 六、修改数据

### 修改店铺信息
打开 `miniprogram/app.js`，找到 `globalData`：

```javascript
globalData: {
  storeInfo: {
    name: '你的店铺名称',           // 修改店铺名称
    address: '你的店铺地址',        // 修改地址
    phone: '400-888-8888',         // 修改电话
    mobile: '138-0000-0000',       // 修改手机号
    hours: '周一至周日 09:00 - 18:00'  // 修改营业时间
  }
}
```

### 修改价格数据
打开 `miniprogram/app.js`，找到 `updatePriceData` 方法：

```javascript
{
  id: 1,
  code: 'gold_9999',
  name: '黄金9999',
  buyPrice: 558.50,    // 修改回购价
  sellPrice: 568.80,   // 修改销售价
  highPrice: 570.20,   // 修改最高价
  lowPrice: 556.30     // 修改最低价
}
```

保存后，按 `Ctrl+S` (Windows) 或 `Cmd+S` (Mac)，小程序会自动刷新。

## 八、常见问题

### Q: 导入时提示"不是有效的小程序目录"？
**A**: 请确保选择的是 `miniprogram` 文件夹，不是 `mini-goldprice` 根目录。

### Q: TabBar 没有图标？
**A**: 正常！我们使用文字+Emoji的方式，不需要图标图片。

### Q: 如何修改价格？
**A**: 编辑 `miniprogram/app.js` 文件，找到 `updatePriceData` 方法修改价格数据。

### Q: 可以在手机上看吗？
**A**: 可以！点击"预览"按钮，用微信扫码即可在手机上查看。

### Q: AppID 是什么？
**A**: 小程序的唯一标识。开发测试时选择"测试号"即可，正式上线需要申请。
