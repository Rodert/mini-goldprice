## 🚀 使用方法

### 方式一：立即测试（使用模拟数据）

```bash
# 1. 启动后端
cd gold-admin-backend
go run main.go

# 2. 启动前端
cd gold-admin-frontend
npm run serve

# 3. 访问页面
# 浏览器打开: http://localhost:8080/display
# 按 F11 进入全屏
```

### 方式二：完整体验（添加资源文件）

## 📊 技术实现

### 前端技术
- **Vue 2**: 组件化开发
- **Element UI**: UI组件（图标等）
- **SCSS**: 样式预处理
- **CSS3 Animation**: 流畅动画
- **HTML5 Video**: 视频播放
- **Axios**: API请求

### 关键技术点
1. **无缝滚动**: 通过复制数组实现无缝循环
2. **性能优化**: 使用 `transform` 代替 `position` 实现硬件加速
3. **时间更新**: `setInterval` 定时器
4. **视频控制**: HTML5 Video API
5. **响应式**: Flexbox 布局
