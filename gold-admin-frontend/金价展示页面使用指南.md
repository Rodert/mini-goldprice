# 金价展示页面使用指南

## 功能概述

金价展示页面是一个专门用于店铺电子屏展示的全屏页面，包含以下功能：

1. **顶部横幅**
   - 左侧：品牌Logo和名称（SineGem 中国珠宝）
   - 右侧：今日金价标题

2. **主体内容区（三栏布局）**
   - **左侧**：珠宝宣传视频循环播放
   - **中间**：产品图片垂直无缝滚动
   - **右侧**：实时金价表格（从后台API自动获取）

3. **底部信息栏**
   - 左侧：实时日期时间显示
   - 中间：视频播放进度条
   - 右侧：欢迎语横向滚动（跑马灯效果）

## 快速开始

### 1. 准备资源文件

在 `gold-admin-frontend/public/assets/` 目录下添加以下文件：

- `jewelry-video.mp4` - 珠宝宣传视频（必需）
- `product1.jpg` ~ `product5.jpg` - 产品展示图片（至少5张）

> 资源文件详细要求请参考 `public/assets/README.md`

### 2. 启动服务

```bash
cd gold-admin-frontend
npm run serve
```

### 3. 访问页面

浏览器打开：`http://localhost:8080/display`

**无需登录**即可访问该页面！

## 自定义配置

### 修改滚动速度

编辑 `src/views/display/index.vue`，找到以下配置：

```javascript
scrollSpeed: 30,    // 产品图片滚动速度（像素/秒）
marqueeSpeed: 100,  // 欢迎语滚动速度（像素/秒）
```

### 修改刷新频率

找到以下代码修改金价刷新频率：

```javascript
// 定期刷新价格数据（每5分钟）
this.refreshTimer = setInterval(() => {
  this.fetchPriceData()
}, 5 * 60 * 1000)  // 修改这里的数值（毫秒）
```

### 修改品牌信息

在模板中找到以下代码修改：

```vue
<div class="logo">
  <span class="logo-icon">💎</span>
  <span class="brand-name">SineGem 中国珠宝</span>  <!-- 修改这里 -->
</div>
<div class="title">今日金价</div>  <!-- 修改这里 -->
```

### 修改欢迎语

找到以下代码修改跑马灯文字：

```vue
<div class="marquee-text" :style="{ transform: `translateX(${marqueeX}px)` }">
  中国珠宝欢迎您！！！  <!-- 修改这里 -->
</div>
```

## 常见问题

### Q: 视频不显示怎么办？
A: 检查 `public/assets/jewelry-video.mp4` 文件是否存在，确保视频格式为 MP4 (H.264编码)

### Q: 产品图片不滚动？
A: 检查 `public/assets/product*.jpg` 文件是否存在，至少需要5张图片

### Q: 金价数据不显示？
A: 
1. 确保后端服务正常运行
2. 检查浏览器控制台是否有API错误
3. 如果API失败，会自动显示模拟数据

### Q: 如何设置开机自动启动？
A: 
1. 设置浏览器开机自启动
2. 设置浏览器主页为 `http://localhost:8080/display`
3. 或使用 kiosk 模式启动浏览器

### Q: 页面长时间运行会不会卡顿？
A: 页面已优化性能，使用CSS硬件加速，可长时间稳定运行
