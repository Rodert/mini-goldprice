# 金价展示页面功能说明

## 概述

在 `gold-admin-frontend` 中新增了一个独立的金价展示页面，专门用于店铺电子屏、数字标牌等场景。

## 页面特点

### 🎯 核心功能
- **无需登录**：独立页面，不需要管理员权限
- **全屏展示**：适合电子屏、投影、平板等设备
- **实时数据**：自动从后台获取最新金价数据
- **自动动画**：视频循环、图片滚动、跑马灯效果

### 📱 页面布局

```
┌─────────────────────────────────────────────────────────┐
│  💎 SineGem 中国珠宝              今日金价                │
├──────────────┬──────────┬─────────────────────────────┤
│              │          │                             │
│   视频播放   │  产品图  │      金价表格               │
│   (循环)     │  (滚动)  │   品名 | 价格 | 工费        │
│              │          │                             │
│              │          │                             │
├──────────────┴──────────┴─────────────────────────────┤
│ 2025-11-05 星期三  |  进度条  |  中国珠宝欢迎您>>>     │
└─────────────────────────────────────────────────────────┘
```

### ✨ 动画效果
1. **左侧**：珠宝宣传视频自动循环播放
2. **中间**：产品图片垂直无缝滚动
3. **右侧**：金价表格实时显示（从API获取）
4. **底部**：
   - 实时日期时间
   - 视频播放进度
   - 欢迎语横向滚动（跑马灯）

## 快速开始

### 1. 准备资源文件（可选）

在 `gold-admin-frontend/public/assets/` 目录下添加：
- `jewelry-video.mp4` - 珠宝宣传视频
- `product1.jpg` ~ `product5.jpg` - 产品图片

> 如果暂时没有资源，页面会显示占位符，不影响功能测试

### 2. 启动服务

```bash
# 启动后端（在 gold-admin-backend 目录）
cd gold-admin-backend
go run main.go

# 启动前端（在 gold-admin-frontend 目录）
cd gold-admin-frontend
npm run serve
```

### 3. 访问页面

浏览器打开：`http://localhost:8080/display`

按 `F11` 进入全屏模式获得最佳效果

## 数据来源

### 金价数据
- 从后台 API 自动获取：`GET /api/v1/prices`
- 每5分钟自动刷新一次
- 如果API失败，自动使用模拟数据

### 配置项
在 `src/views/display/index.vue` 中可以修改：
- 滚动速度
- 刷新频率
- 品牌信息
- 欢迎语内容

详细配置请查看 **金价展示页面使用指南.md**
