# 快速测试指南

## 本地测试

### 1. 使用 Python 简单服务器（推荐）

```bash
cd gold-display-standalone

# Python 3
python3 -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然后在浏览器中打开：`http://localhost:8000`

### 2. 使用 Node.js http-server

```bash
# 安装 http-server（如果还没有）
npm install -g http-server

# 启动服务
cd gold-display-standalone
http-server -p 8000
```

### 3. 使用 PHP 内置服务器

```bash
cd gold-display-standalone
php -S localhost:8000
```

## 常见问题

### Q: 页面显示空白

**A:** 检查：
1. 浏览器控制台是否有错误
2. 文件路径是否正确
3. 服务器是否正常运行

### Q: API 请求失败

**A:** 检查：
1. 后端服务是否运行
2. `apiBaseUrl` 配置是否正确
3. 是否跨域（需要 CORS 支持）

### Q: 视频无法播放

**A:** 检查：
1. 视频文件是否存在
2. 视频格式是否为 MP4
3. 浏览器是否支持 HTML5 video

### Q: 图片无法显示

**A:** 检查：
1. 图片路径是否正确
2. 图片文件是否存在
3. 网络连接是否正常（如果使用在线图片）

## 调试技巧

### 1. 查看控制台日志

打开浏览器开发者工具（F12），查看 Console 标签，所有日志都会显示在这里。

### 2. 检查网络请求

在 Network 标签中查看：
- API 请求是否发送
- 请求状态码
- 响应内容

### 3. 检查元素

在 Elements 标签中检查：
- DOM 结构是否正确
- 样式是否应用
- 元素是否存在

## 性能测试

### 1. 检查加载时间

在 Network 标签中查看页面加载时间，应该 < 2 秒。

### 2. 检查内存使用

在 Performance 标签中记录页面运行，检查内存使用是否稳定。

### 3. 检查动画流畅度

观察动画是否流畅，60fps 为最佳。
